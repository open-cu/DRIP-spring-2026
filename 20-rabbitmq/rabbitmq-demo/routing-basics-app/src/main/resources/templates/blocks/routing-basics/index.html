Блок routing-basics

Тренируем 4 типа маршрутизации в RabbitMQ: direct, topic, fanout, headers. Отправляем сообщение и сразу смотрим изменение статистики очередей.

Шпаргалка по маршрутизации

  • Direct: точное совпадение routing key и binding key.
  • Topic: шаблон ключа: * — одно слово, # — много слов.
  • Fanout: broadcast во все связанные очереди, ключ игнорируется.
  • Headers: маршрутизация по заголовкам, в примере — X-Priority.
